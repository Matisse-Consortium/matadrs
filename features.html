Features#

For working examples of all the features, and more explanations on the usage see Getting Started and for an in depth view of the individual code, see API.

Raw Data Retrieval#

Automatic download of the data for one or more specific night(s) from the ESO-archives.

Yet to be implemented

Reduction#

The reduction can be done with simply using the function matadrs.matadrs.reduction_pipeline().

The function matadrs.matadrs.reduction_pipeline() combines functionality from the following modules (found under the matadrs.reduction subpackage).

  • matadrs.reduction.reduce

  • matadrs.reduction.calibrate

  • matadrs.reduction.average

  • matadrs.reduction.merge

These modules take care of the reduction, calibration of the fluxes and visibilities, averaging all datasets together, and finally merging all of them into one final, oifits-compliant (.fits)-file.

Readout#

This enables the user to read in a (.fits)-file and get the data in the form of a astropy.table for every header in the (.fits)-file (oi_vis, oi_t3, etc.), as long as the file is in the oifits format.

Plotting#

There is additional plotting functionality included in matadrs in order to immediately visualise the products of the data-reduction pipeline at each individual step (see Reduction).
